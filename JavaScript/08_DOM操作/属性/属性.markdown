## 基础概念

- 属性在 HTML 或 XML 中，标签可能包含属性，这些属性提供关于元素的额外信息。例如，在 `<a href="https://example.com">链接</a>` 这个标签中，`href` 就是一个属性，它的值为 `"https://example.com"`。
